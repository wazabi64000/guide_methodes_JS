Répertoire des Méthodes JavaScript

Méthode Description Exemples
sort() Trie les éléments d’un tableau selon un ordre donné.
Simple :
[3,1,2].sort();
Intermédiaire :
["b","a","c"].sort();
Avancé :
const users = [{n:"Paul",a:30},{n:"Anna",a:20}]; users.sort((x,y) => x.a - y.a);
slice() Retourne une copie partielle d’un tableau ou une chaîne.
Simple :
[1,2,3,4].slice(1,3);
Intermédiaire :
"Bonjour".slice(0,4);
Avancé :
const items = [...Array(10).keys()]; items.slice(-3);
trim() Supprime les espaces en début et fin de chaîne.
Simple :
" test ".trim();
Intermédiaire :
[" a "," b "].map(x => x.trim());
Avancé :
const clean = str => str.trim().replace(/\s+/g," "); clean(" Hello world ");
map() Transforme chaque élément d’un tableau et retourne un nouveau tableau.
Simple :
[1,2,3].map(n => n * 2);
Intermédiaire :
["a","b"].map((c,i) => i + "-" + c);
Avancé :
const users = [{n:"A",points:10},{n:"B",points:20}]; users.map(u => ({...u,bonus:u.points*0.2}));
filter() Retourne les éléments d’un tableau qui satisfont une condition.
Simple :
[1,2,3,4].filter(n => n > 2);
Intermédiaire :
["test","","ok"].filter(x => x);
Avancé :
const list = [{v:10},{v:0},{v:5}]; list.filter(o => o.v >= 5);
reduce() Accumule les valeurs d’un tableau selon une logique donnée.
Simple :
[1,2,3].reduce((a,b) => a + b, 0);
Intermédiaire :
["a","b","c"].reduce((s,c) => s + "-" + c);
Avancé :
const cart = [{p:20},{p:10},{p:15}]; cart.reduce((total, item) => total + item.p, 0);
forEach() Exécute une fonction sur chaque élément sans retourner de valeur.
Simple :
[1,2].forEach(n => console.log(n));
Intermédiaire :
const out = []; [1,2,3].forEach(n => out.push(n*2));
Avancé :
const users = ["Anna","Paul"]; users.forEach((u,i) => console.log(i+1, u));
find() Retourne le premier élément correspondant à une condition.
Simple :
[1,4,6].find(n => n > 3);
Intermédiaire :
["A","B","C"].find(c => c === "B");
Avancé :
const users = [{id:1},{id:2},{id:3}]; users.find(u => u.id === 2);
includes() Vérifie si un tableau ou une chaîne contient une valeur.
Simple :
[1,2,3].includes(2);
Intermédiaire :
"bonjour".includes("jour");
Avancé :
const keys = ["admin","user","guest"]; keys.includes("admin");
push() Ajoute un élément à la fin d’un tableau.
Simple :
const a=[1]; a.push(2);
Intermédiaire :
const list=[1,2]; [3,4].forEach(n => list.push(n));
Avancé :
const users=[]; const addUser = name => users.push({name}); addUser("Paul");
pop() Supprime et retourne le dernier élément d’un tableau.
Simple :
[1,2,3].pop();
Intermédiaire :
const a=[5,6,7]; const last=a.pop();
Avancé :
const stack=[1,2,3]; while(stack.length) stack.pop();
shift() Supprime et retourne le premier élément du tableau.
Simple :
[1,2,3].shift();
Intermédiaire :
const q=[10,20,30]; const first=q.shift();
Avancé :
const queue=[1,2,3]; while(queue.length) queue.shift();
unshift() Ajoute un ou plusieurs éléments au début.
Simple :
[2,3].unshift(1);
Intermédiaire :
const a=[3,4]; a.unshift(1,2);
Avancé :
const events=[]; events.unshift({type:"click",t:Date.now()});
splice() Supprime, remplace ou ajoute des éléments dans un tableau.
Simple :
[1,2,3].splice(1,1);
Intermédiaire :
const a=[1,2,3,4]; a.splice(2,0,"X");
Avancé :
const items=["a","b","c"]; items.splice(1,1,"new","value");
concat() Fusionne des tableaux ou des chaînes.
Simple :
[1,2].concat([3,4]);
Intermédiaire :
"Hello ".concat("World");
Avancé :
const all=[...a1.concat(a2).concat(a3)];
join() Convertit un tableau en chaîne avec un séparateur.
Simple :
[1,2,3].join("-");
Intermédiaire :
["a","b","c"].join("");
Avancé :
const path=["usr","local","bin"]; path.join("/");
reverse() Inverse l’ordre des éléments d’un tableau.
Simple :
[1,2,3].reverse();
Intermédiaire :
"abc".split("").reverse().join("");
Avancé :
const logs=[3,2,1]; const sorted=logs.reverse().map(x=>x*x);
indexOf() Retourne l’indice de la première occurrence.
Simple :
[1,2,3].indexOf(2);
Intermédiaire :
"banana".indexOf("na");
Avancé :
const keys=["id","name","age"]; keys.indexOf("age");
lastIndexOf() Retourne la dernière occurrence dans un tableau ou une chaîne.
Simple :
[1,2,2,3].lastIndexOf(2);
Intermédiaire :
"banana".lastIndexOf("na");
Avancé :
const nums=[5,7,5,9]; nums.lastIndexOf(5);
includes() Indique si un tableau contient une valeur donnée.
Simple :
[1,2,3].includes(3);
Intermédiaire :
"javascript".includes("script");
Avancé :
const permissions=["read","write"]; permissions.includes("write");
flat() Aplati les sous-tableaux selon un niveau donné.
Simple :
[1,[2,3]].flat();
Intermédiaire :
[1,[2,[3]]].flat(2);
Avancé :
const arr=[1,[2],[3,[4]]]; arr.flat(Infinity);
flatMap() Map + flat en un seul passage.
Simple :
[1,2].flatMap(x => [x,x*2]);
Intermédiaire :
["a","b"].flatMap(c => [c.toUpperCase(),c]);
Avancé :
const users=[{id:1,tags:["js","html"]},{id:2,tags:["css"]}]; users.flatMap(u => u.tags);
split() Divise une chaîne selon un séparateur.
Simple :
"a-b-c".split("-");
Intermédiaire :
"hello world".split(" ");
Avancé :
"1,2;3|4".split(/[,;|]/);
replace() Remplace la première occurrence dans une chaîne.
Simple :
"abc".replace("a","x");
Intermédiaire :
"Hello Bob".replace("Bob","Alice");
Avancé :
"2025-11-18".replace(/-/,"/");
replaceAll() Remplace toutes les occurrences dans une chaîne.
Simple :
"aaa".replaceAll("a","b");
Intermédiaire :
"test test".replaceAll("t","T");
Avancé :
"10-20-30".replaceAll(/-/g,":");
startsWith() Vérifie si une chaîne commence par un motif.
Simple :
"abc".startsWith("a");
Intermédiaire :
"bonjour".startsWith("bon");
Avancé :
const url="https://site.com"; url.startsWith("https");
endsWith() Vérifie si une chaîne se termine par un motif.
Simple :
"abc".endsWith("c");
Intermédiaire :
"photo.jpg".endsWith(".jpg");
Avancé :
const token="user-12345"; token.endsWith("12345");
padStart() Ajoute des caractères au début d’une chaîne jusqu’à atteindre une longueur donnée.
Simple :
"5".padStart(3,"0");
Intermédiaire :
"42".padStart(5,".");
Avancé :
const id=12; String(id).padStart(6,"0");
padEnd() Ajoute des caractères à la fin d’une chaîne.
Simple :
"5".padEnd(3,"0");
Intermédiaire :
"JS".padEnd(6,"*");
Avancé :
const name="Paul"; name.padEnd(10," ");
repeat() Répète une chaîne selon un nombre donné.
Simple :
"ha".repeat(3);
Intermédiaire :
"*".repeat(10);
Avancé :
const banner="-".repeat(50);
charAt() Retourne le caractère à un index donné.
Simple :
"Hello".charAt(1);
Intermédiaire :
"Bonjour".charAt("Bonjour".length - 1);
Avancé :
function safeChar(str,i){ return str.charAt(i) || ""; } safeChar("Test",50);
charCodeAt() Retourne le code UTF-16 du caractère.
Simple :
"A".charCodeAt(0);
Intermédiaire :
"abc".charCodeAt(2);
Avancé :
"AZ".split("").map(l => l.charCodeAt(0));
fromCharCode() Convertit des codes UTF-16 en caractères.
Simple :
String.fromCharCode(65);
Intermédiaire :
String.fromCharCode(72,101,108,108,111);
Avancé :
[65,66,67].map(c => String.fromCharCode(c)).join("");
localeCompare() Compare deux chaînes selon la locale.
Simple :
"a".localeCompare("b");
Intermédiaire :
"école".localeCompare("ecole","fr");
Avancé :
["z","é","a"].sort((a,b)=>a.localeCompare(b,"fr"));
match() Retourne les correspondances d’une expression régulière.
Simple :
"abc123".match(/[0-9]/);
Intermédiaire :
"hello 2025".match(/\d+/);
Avancé :
"js-html-css".match(/[a-z]+/g);
matchAll() Retourne un itérateur de toutes les correspondances.
Simple :
[... "a1b2".matchAll(/\d/)];
Intermédiaire :
[..."ab12cd34".matchAll(/\d+/g)];
Avancé :
const reg=/([a-z]+)(\d+)/g; [..."js12css34".matchAll(reg)];
search() Retourne l’index de la première correspondance.
Simple :
"hello".search(/e/);
Intermédiaire :
"javascript".search(/script/);
Avancé :
"file-2025-version".search(/\d{4}/);
concat() Fusionne des chaînes ou tableaux.
Simple :
[1,2].concat([3]);
Intermédiaire :
"Hello".concat(" ","World");
Avancé :
const base=[1,2]; const extra=[3,4]; base.concat(extra,["x"]);
reduceRight() Équivaut à reduce mais de droite à gauche.
Simple :
[1,2,3].reduceRight((a,b)=>a+b);
Intermédiaire :
["a","b","c"].reduceRight((a,b)=>a+b);
Avancé :
[[1],[2],[3]].reduceRight((acc,v)=>acc.concat(v),[]);
every() Vérifie si tous les éléments respectent une condition.
Simple :
[2,4,6].every(n=>n%2===0);
Intermédiaire :
["a","b"].every(x=>typeof x==="string");
Avancé :
const users=[{admin:true},{admin:false}]; users.every(u=>u.admin===true);
some() Vérifie si au moins un élément respecte une condition.
Simple :
[1,3,5].some(n=>n%2===0);
Intermédiaire :
["a",2,"c"].some(x=>typeof x==="number");
Avancé :
const users=[{role:"user"},{role:"admin"}]; users.some(u=>u.role==="admin");
fill() Remplit un tableau par une valeur.
Simple :
new Array(3).fill(0);
Intermédiaire :
[1,2,3,4].fill("x",1,3);
Avancé :
Array(5).fill().map((_,i)=>i+1);
includes() Vérifie si une valeur est présente dans un tableau ou une chaîne.
Simple :
[1,2,3].includes(2);
Intermédiaire :
"javascript".includes("script");
Avancé :
const roles=["user","manager","admin"]; roles.includes("admin");
toUpperCase() Convertit une chaîne en majuscules.
Simple :
"hello".toUpperCase();
Intermédiaire :
"école".toUpperCase();
Avancé :
["js","css","html"].map(s=>s.toUpperCase());
toLowerCase() Convertit une chaîne en minuscules.
Simple :
"HELLO".toLowerCase();
Intermédiaire :
"ÉCOLE".toLowerCase();
Avancé :
["JS","CSS","HTML"].map(s=>s.toLowerCase());
trim() Supprime les espaces inutiles autour d’une chaîne.
Simple :
" test ".trim();
Intermédiaire :
" hello world".trim();
Avancé :
[" a "," b "," c "].map(s=>s.trim());
trimStart() Supprime uniquement les espaces au début.
Simple :
" test".trimStart();
Intermédiaire :
" hello ".trimStart();
Avancé :
[" js"," css"].map(s=>s.trimStart());
trimEnd() Supprime les espaces à la fin d’une chaîne.
Simple :
"test ".trimEnd();
Intermédiaire :
"hello ".trimEnd();
Avancé :
["js ","css "].map(s=>s.trimEnd());
parseInt() Convertit une chaîne en entier.
Simple :
parseInt("42");
Intermédiaire :
parseInt("1010",2);
Avancé :
["10","20px","30.9"].map(v=>parseInt(v));
parseFloat() Convertit une chaîne en nombre décimal.
Simple :
parseFloat("3.14");
Intermédiaire :
parseFloat("42.55px");
Avancé :
["1.1","2.2","3.33px"].map(v=>parseFloat(v));
