map()Transforme chaque élément → nouveau tableau.
1. Âge + 2030
const amis = [{nom:"Perrine",age:28},{nom:"Jacqueline",age:34},{nom:"Lucie",age:25},{nom:"Romain",age:31}];
amis.map(a => a.age + 5); // → [33,39,30,36]
2. Prénoms en majuscules
amis.map(a => a.nom.toUpperCase()); // ["PERRINE","JACQUELINE",...]
3. Cartes HTML
amis.map(a => `
${a.nom} – ${a.age} ans
`).join("")
filter()Garde les éléments qui passent le test.
1. Plus de 30 ans
amis.filter(a => a.age > 30); // Jacqueline & Romain
2. Prénom avec "L"
amis.filter(a => a.nom.includes("L")); // Lucie
3. Nettoyage
["Perrine","","Lucie",null].filter(Boolean); // ["Perrine","Lucie"]
