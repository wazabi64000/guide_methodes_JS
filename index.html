JavaScript Simple & Complet

53 méthodes • Perrine • Jacqueline • Lucie • Romain

1. map()

Transforme chaque élément → nouveau tableau
const amis = [{nom:"Perrine",age:28},{nom:"Jacqueline",age:34},{nom:"Lucie",age:25},{nom:"Romain",age:31}];

amis.map(a => a.nom)
// → ["Perrine","Jacqueline","Lucie","Romain"]

[10,15,20].map(x => x*2)
// → [20,30,40]

amis.map(a => `<div>${a.nom} (${a.age} ans)</div>`)

2. filter()

Garde seulement ce qui passe le test
amis.filter(a => a.age >= 30)
// → Jacqueline, Romain

amis.filter(a => a.nom.startsWith("L"))
// → ["Lucie"]

["Perrine","","Lucie",null].filter(Boolean)
// → ["Perrine","Lucie"]

3. reduce()

Transforme tout en une valeur
amis.reduce((s,a) => s + a.age, 0) / amis.length
// → 29.5

amis.reduce((g,a) => {
  const t = a.age < 30 ? "jeunes" : "seniors";
  g[t] ||= []; g[t].push(a.nom); return g;
}, {})
// → {jeunes:["Perrine","Lucie"], seniors:["Jacqueline","Romain"]}

4. find()

Premier élément qui correspond
amis.find(a => a.nom === "Romain")
// → {nom:"Romain", age:31}

amis.find(a => a.age < 26)
// → {nom:"Lucie", age:25}

5. findIndex()

Index du premier trouvé (-1 si absent)
const i = amis.findIndex(a => a.nom === "Lucie");
// → 2
if(i > -1) amis.splice(i,1); // supprime Lucie

6. some()

Au moins un ?
amis.some(a => a.age > 33)     // true (Jacqueline)
amis.some(a => a.nom === "Perrine") // true

7. every()

Tous ?
amis.every(a => a.age >= 18)   // true

8. includes()

Existe dans tableau/chaîne
["Perrine","Lucie"].includes("Lucie") // true
"Perrine".includes("rr") // true

9. forEach()

Exécute pour chacun
amis.forEach(a => console.log(a.nom))

10. sort()

Trie en place
amis.sort((a,b) => a.age - b.age)
// → Lucie, Perrine, Romain, Jacqueline

11. slice()

Copie une partie
amis.slice(-2) // Romain, Jacqueline
amis.slice()   // clone complet

12. splice()
