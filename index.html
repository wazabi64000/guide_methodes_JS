map()Transforme chaque élément → nouveau tableau de même longueur. Immuable. Indispensable en React/Vue.
1. Doubler
const n = [1,2,3]; const d = n.map(x => x*2); // → [2,4,6]
2. Extraire noms
users.map(u => u.nom); // → ["Alice","Bob"]
3. HTML React-like
items.map((i,idx) => `
  • ${i}
    • `);
    filter()Ne garde que les éléments où la fonction retourne true. Immuable. Souvent chaîné avec map().
    1. Pairs
    [1,2,3,4].filter(x => x%2===0); // → [2,4]
    2. Majeurs
    users.filter(u => u.age >= 18);
    3. Supprimer falsy
    arr.filter(Boolean); // magique !
    reduce()La méthode la plus puissante : transforme un tableau en n’importe quoi (somme, objet, chaîne…).
    1. Somme
    [10,20,30].reduce((a,b)=>a+b, 0); // → 60
    2. Compter
    votes.reduce((a,v)=>(a[v]=(a[v]||0)+1,a), {});
    3. Grouper
    personnes.reduce((g,p)=>{ g[p.ville] ||= []; g[p.ville].push(p); return g; }, {});
    find()Retourne le premier élément qui passe le test (ou undefined). Arrête dès qu’il trouve.
    1. Par ID
    users.find(u => u.id === 42);
    2. Premier négatif
    [10,-5,20].find(x=>x<0); // → -5
    3. Avec fallback
    findAdmin() || {role:"guest"};
    findIndex()Retourne l’index du premier élément trouvé, ou -1.
    1. Supprimer
    const i = arr.findIndex(x=>x.id===5); if(i>-1) arr.splice(i,1);
    2. Mettre à jour
    const i = users.findIndex(u=>u.id===7); if(i>-1) users[i] = nouveau;
    3. Existe ?
    arr.findIndex(x=>x===val) !== -1
    some()true si au moins un élément passe le test.
    1. Admin ?
    users.some(u=>u.role==="admin")
    2. Champ vide
    inputs.some(i=>!i.value.trim())
    3. Autorisation
    roles.some(r=>permissions.includes(r))
    every()true si TOUS les éléments passent le test.
    1. Tous majeurs
    users.every(u=>u.age>=18)
    2. Formulaire valide
    champs.every(c=>c.validité)
    3. Tous positifs
    vals.every(v=>v>0)
